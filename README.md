# SocioCore

![SocioCore Icon](public/img/logo.png)

Simplify Society Management with Our SaaS Solution

An all-in-one platform to efficiently manage apartments, residents and maintenance. Reduce administrative burden while improving operational efficiency.

## Key Features

- **Simplified Resident Management** — Centralize apartment records, ownership details and resident information. Automate rent collection, lease management and resident communications.
- **Optimized Service Provider Coordination** — Manage housekeeping, security and maintenance staff; monitor schedules and track performance.
- **Smart Amenity Management** — Book common facilities (clubhouse, gym, pool) with automated scheduling to prevent double bookings.
- **Smart Billing System** — Automated billing for rent, utilities and maintenance.
- **Secure Payment Processing** — Integrated payments via Stripe and Razorpay.
- **Staff Management** — Individual attendance tracking for service providers.
- **Visitor Access Control** — Digital visitor registration and tracking.
- **Community Updates** — Centralized notice board for communications.
- **Complaint Resolution** — Systematic handling of resident issues.
- **Parking Solutions** — Digital parking space management system.
- **Analytics Dashboard** — Insightful reports for better decisions.

## Quick Start

Copy the example env, install dependencies, migrate the database, and run the dev server:

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm install
npm run dev
php artisan serve
```

## Requirements

- PHP 8.x
- Composer
- Node.js & NPM/Yarn
- Database: MySQL / PostgreSQL
